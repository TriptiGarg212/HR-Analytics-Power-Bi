## Project Overview
This is a beginner-level HR analytics dashboard created using Power BI to analyze employee data and understand workforce trends such as attrition, age distribution, departments, and job satisfaction.

This project was created as a *practice project* to learn Power BI, data cleaning, and dashboard design.

## Tools Used
- Power BI Desktop
- Excel / CSV (HR Dataset)

## Dashboard Preview
A preview of the dashboard is available as a PDF in this repository:
*HR-Analytics-Dashboard.pdf*

## Key Analysis Performed
- Overall employee count and active employees
- Employee attrition count and attrition rate
- Attrition analysis by department and education field
- Employee distribution by age group and gender
- Job satisfaction analysis across job roles

## Learning Outcomes
- Learned how to load and clean data using Power Query
- Created KPI cards, bar charts, pie charts, and tables in Power BI
- Understood basic HR metrics and how to represent them visually
- Improved dashboard layout and presentation skills
